# PhpStorm

 * Remove trailing whitespace automatically
   1. `File`
   2. `Settings`
   3. `IDE Settings`
   4. `Editor`
   5. Set `Strip trailing spaces on Save` to `All`
 * Make sure that every file ends with a line feed
   1. `File`
   2. `Settings`
   3. `IDE Settings`
   4. `Editor`
   5. Check `Ensure line feed at file end on Save`
